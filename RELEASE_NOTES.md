# Release Notes - StellarPay 1.0.0

**Release Date:** 2026-07-08

We are excited to announce the official 1.0.0 release of StellarPay! This release transforms the application from a basic tutorial prototype into a production-ready Web3 application, successfully achieving all requirements for the Journey to Mastery (White, Yellow, and Orange Belts).

## Key Highlights

- **Multi-Wallet Support:** Users can seamlessly connect their preferred wallets (Freighter, Albedo) via StellarWalletsKit.
- **Smart Contract Receipts:** Integrated Soroban Smart Contracts to store transaction metadata (sender, receiver, amount, memo, timestamp) immutably on the testnet ledger.
- **Live Event Streaming:** Real-time feedback for transaction receipts via Soroban contract events.
- **Flawless UI:** A complete design overhaul utilizing `shadcn/ui`, fully responsive on mobile and desktop devices.
- **Robust CI/CD:** Automated testing, linting, and type checking configured using GitHub Actions.

## Known Limitations
- Currently restricted to the Stellar Testnet. Mainnet support will be added in a future update.
